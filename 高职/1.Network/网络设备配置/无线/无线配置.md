## 无线配置

- 为管理和业务网段配置dhcp

- 交换机与ac间互联口使用trunk

  

- 配置ac

```shell

ac-controller //进入ac配置模式

ac-name Beijing-AC //配置ac名称

capwap ctrl-ip 172.16.100.1 //配置隧道管理地址

```

  

- 为ap配置dhcp地址

```shell

ip dhcp pool 10

option 138 ip 172.16.100.1 172.16.100.2 //配置capwap隧道发现ac

network 172.16.1.0 255.255.255.128

default-router 172.16.1.3

  

#（使用show ap-config summary查看ap注册情况）

ap-group BJ //创建ap组

ap-config MAC地址 //进入ap配置模式

ap-name xxxx //更改ap名称

ap-group BJ //绑定所在组

  

response-rssi 30 radio 1 //配置 RSSI值30对应的信号强度为-65dBm

response-rssi 30 radio 2 //30是RSSI的偏移值，对应于-65dBm的信号强度

  

#（使用show ap-config running xxxx查看当前ap所在组）

wlan-config 3 Test-BJ //创建wlan号为3，ssid为Test-BJ

tunnel local //配置该wlan为本地转发模式， 让接入交换机决定终端ip

ap-group BJ

interface-mapping 3 20 ap-wlan-id 1 //将3的wlan-id映射到ap组内的所有ap的radio上，20为无线业务vlan，最后的1为默认配置

wlansec 3 //进入wlan的安全配置模式

security wpa enable //配置启用wpa认证加密模式

security wpa ciphers aes enable //配置 WPA 认证模式的数据加密方式为 AES

security wpa akm psk enable //配置 WPA 认证模式的接入认证方式为 PSK

security wpa akm psk set-key ascii 12345678 //配置 PSK 密码为 12345678

  
  

wlan-based per-user-limit down-streams average-data-rate 1000 burst-data-rate 2000 //将速率单位转换为8Kbps，即800Kbps等于1000，1600Kbps等于2000

  
  

Ruijie(config-wlansec)#security rsn enable

Ruijie(config-wlansec)#security rsn ciphers aes enable

Ruijie(config-wlansec)#security rsn akm psk enable

Ruijie(config-wlansec)#security rsn akm psk set-key ascii 12345678 //配置wpa2（rsn）认证模式

  

#配置ac虚拟化

virtual-ac domain //配置域 ID

device //配置设备在虚拟设备中的编号

device priority //配置设备的优先级

device connect_switch //配置 AC 连接的交换机设备 id

vac-port //进入心跳链路端口配置模式

port-member interface //把普通口配置到心跳链路端口池中

device convert mode vitual //单机模式切换到 VAC 模式

  

#配置ac的wlan热备份

#当主 AC 设备不可达（例如 AC 升级重启、机房停电等原因造成）的情况下，备 AC 设备能够接替主 AC 设备工作，保证无线

  

#网络不受影响

wlan hot-backup //配置热备邻居，进入热备份配置界面

wlan hot-backup enable //开启热备功能

local-ip ip-address //设置热备的本端 IP 地址（默认为回环0）

context //配置热备实例，进入实例配置界面

priority level //配置热备实例优先级

ap-group ap-group //绑定需要热备份的ap组

```

#禁用无线网络中的低速率

```shell

802.11b network rate 1 disabled // 禁用1M速率

802.11b network rate 2 disabled // 禁用2M速率

802.11b network rate 5 disabled // 禁用5.5M速率

802.11g network rate 1 disabled // 禁用1M速率

802.11g network rate 2 disabled // 禁用2M速率

802.11g network rate 5 disabled // 禁用5.5M速率

802.11g network rate 6 disabled // 禁用6M速率

802.11g network rate 9 disabled // 禁用9M速率

802.11a network rate 6 disabled // 禁用6M速率

802.11a network rate 9 disabled // 禁用9M速率

```
