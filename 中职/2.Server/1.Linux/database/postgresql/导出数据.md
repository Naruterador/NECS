## 导出数据表字段，用逗号隔开
```postgresql
#创建root用户
#登录到 PostgreSQL 用户
sudo -u postgres psql

#创建root用户
CREATE USER root WITH PASSWORD 'your_password';

#提升root用户为超级管理员
ALTER USER root SUPERUSER;

#授予 "root" 用户适当的权限
GRANT ALL PRIVILEGES ON DATABASE userdb TO root;

#登录数据库
psql -U root -d database

#导出数据表
\copy userinfo to /opt/postgresql/userinfo.sql with delimiter ',';

//在shell下操作
psql -U root -d userdb -t -A -F',' -c "SELECT * FROM userinfo" > /var/pgbak/userinfo.sql

```


#### 导出数据库
```postgresql
//导出数据库代码格式如下:
pg_dump dbname > dumpfile

pg_dump -U root -d userdb > /var/local/postgresqlbak/userdb.sql
```
