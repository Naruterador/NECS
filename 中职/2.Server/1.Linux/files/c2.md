## 题目
  在/tmp/skills_tmp目录下创建File01~File20共20个文件，修改File01~File05文件的最近更改时间为2020-12-19 12:19:00，修改File06~File10文件的最近访问时间为2022-12-19 12:19:00，使用/etc/passwd的时间修改File11~File15文件的最近访问及更改时间。
  
## 答案
```shell
# 创建文件
mkdir -p /tmp/skills_tmp && cd /tmp/skills_tmp
touch File{01..20}

# 修改 File01~File05 文件的修改时间为 2020-12-19 12:19:00
touch -m -d "2020-12-19 12:19:00" File{01..05}

# 修改 File06~File10 文件的访问时间为 2022-12-19 12:19:00
touch -a -d "2022-12-19 12:19:00" File{06..10}

# 获取 /etc/passwd 文件的时间，并用它来修改 File11~File15 的访问和修改时间
touch -r /etc/passwd File{11..15}

#使用 stat 命令查看文件的时间：
stat filename


touch参数说明
-t :指定文件的时间戳（修改时间或访问时间）为用户提供的时间
-a :只修改文件的访问时间（atime），而不改变文件的修改时间（mtime）
-m :选项仅修改修改时间
-d :指定新的时间
-r :参数复制参考文件的时间戳到目标文件
```
